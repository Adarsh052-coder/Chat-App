A lightweight, real-time chat application demonstrating instant messaging using WebSockets. Built with React for the frontend and Express.js for the backend.

Features
.Real-time messaging with WebSocket connections
.Multiple users can chat simultaneously
.Clean architecture with separate frontend and backend
.Minimal design focused on core functionality

Tech Stack
.Frontend: React.js
.Backend: Express.js, WebSocket (ws library)
.Protocol: WebSocket for full-duplex communication

Getting Started
.Clone the repository
.Navigate to the backend directory and install dependencies
.Start the backend server (runs on ws://localhost:8080)
.Navigate to the frontend directory and install dependencies
.Start the React development server (opens at http://localhost:3000)
.Open multiple browser tabs at http://localhost:3000
.Enter a username when prompted and start chatting

Usage
Open multiple browser windows or tabs, enter a username, and send messages. All connected users will see messages instantly, demonstrating real-time bidirectional communication.

Use Cases
.Learning WebSocket implementation
.Building live chat features
.Real-time collaboration tools
.Group messaging prototypes

Purpose
This project serves as a practical example for understanding real-time communication, WebSocket connections, and integrating React with Express.js in a simple, extensible architecture.
